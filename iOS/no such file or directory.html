iOS no such file or directory

[疑难杂症]

在运行项目的时候突然报错:
clang: error: no such file or directory: 'SystemConfiguration'
Command Ld failed with a nonzero exit code

系统库怎么会找不到呢?

通过比对之后, 发现如下解决方法:

build settings -> other Linker Flags 找到 "SystemConfiguration" 并选中. 然后点击 ＋ 号, 输入: -framework.

问题是因为 "SystemConfiguration" 上面少了 -framework 所以出错.

具体更深层次的原因, 不太明白.
