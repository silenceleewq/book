iOS 获取手机型号以及系统版本号

手机型号是可以从 Wikipedia 上面查到的, 这个查到的是手机内部的model和手机型号的对应关系. 比如 iPhone XS 的 model 值是 iPhone11,2 地址: https://www.theiphonewiki.com/wiki/Models 通过代码获取(更多的映射代码,放在了文章最后面): 

- (NSString*)iphoneType {

  //需要导入头文件：#import <sys/utsname.h>
  //https://www.jianshu.com/p/02bba9419df8

  struct utsname systemInfo;

  uname(&systemInfo);

  NSString *platform = [NSString
                        stringWithCString:systemInfo.machine
                        encoding:NSASCIIStringEncoding];
  if ([platform isEqualToString:@"iPhone12,1"]) return @"iPhone 11";

  if ([platform isEqualToString:@"iPhone12,3"]) return @"iPhone 11 Pro";

  if ([platform isEqualToString:@"iPhone12,5"]) return @"iPhone 11 Pro Max";

  return platform;
}

获取手机系统版本号:

NSString *strSysVersion = [[UIDevice currentDevice] systemVersion];
