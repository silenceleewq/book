node 获取终端参数

在 WebStorm 中, 首先创建一个 shell 文件, 内容添加如下:

#! /bin/bash
node ./obtainParameter.js hello world

使用 node 命令执行 ./obtainParameter.js,

在 obtainParameter.js 里面, 添加如下内容:

let args = process.argv
console.log(args)

// 通过 [] 将参数进行解包.
let [arg1, arg2] = process.argv.slice(2)
console.log(arg1)
console.log(arg2);

输出结果为:

[ '/usr/local/bin/node',
  '/Users/lirenqiang1/code_template/JavaScript/JSStudy/Test/obtainParameter.js',
  'hello',
  'world' ]
hello
world
